6.3 Servidores en red

Tipos de servidores esenciales y cómo configurarlos en Windows y Ubuntu.

🌐
Web

🏢
Dominio

🗄️

Windows Configuración de servidores

📁 Servidor de archivosGUI
1
Abre el Administrador del servidor
2
Agregar roles → Servidor de archivos e iSCSI
3
Comparte una carpeta: clic derecho → Propiedades → Compartir
4
Configura permisos en Propiedades → Seguridad → Editar
🏢 Active Directory (dominio)GUI
1
Administrador del servidor → Agregar roles → AD DS
2
Promueve el servidor como controlador de dominio
3
Define el dominio (p. ej., empresa.local)
4
Administra usuarios y grupos desde el Administrador de Active Directory
🌐 Servidor web con IISGUI
1
Administrador del servidor → Agregar roles y características
2
Selecciona el rol Servidor web (IIS) y sigue el asistente
3
Coloca los archivos del sitio en C:\inetpub\wwwroot
4
Inicia el servidor desde el Administrador de IIS
5
Accede al sitio en http://localhost
Prueba rápida: Crea un archivo index.html en C:\inetpub\wwwroot y ábrelo en el navegador con http://localhost.

Ubuntu Configuración de servidores

📁 Servidor de archivos (Samba)CLI
sudo apt install samba
sudo nano /etc/samba/smb.conf
[compartida] path = /ruta/de/la/carpeta read only = no browsable = yes
sudo systemctl restart smbd
Accede desde otro equipo con: \\IP_del_servidor\compartida
🌐 Servidor web (Apache)CLI
sudo apt install apache2
sudo systemctl restart apache2
Coloca los archivos del sitio en /var/www/html. Accede en http://localhost.
🔒 Servidor SSH (acceso remoto)CLI
sudo apt install openssh-server
sudo nano /etc/ssh/sshd_configReforzar seguridad
ssh usuario@ip_del_servidorConectar desde otro equipo
En sshd_config desactiva el acceso root (PermitRootLogin no) y usa claves públicas para mayor seguridad.
🗄️ Servidor de base de datos (MySQL)CLI
sudo apt install mysql-server
sudo mysql_secure_installationAsistente de seguridad
sudo mysql -u root -pAcceder al servidor
