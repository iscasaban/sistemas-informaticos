Windows Herramientas de seguridad

🛡️ Windows DefenderGUI
Acceso
Configuración → Actualización y seguridad → Seguridad de Windows
Funciones principales
✓
Protección en tiempo real contra virus, malware y ransomware
✓
Análisis de amenazas — Protección contra virus → Análisis rápido
🔒 BitLocker (cifrado de disco)GUI
Acceso
Panel de control → Sistema y seguridad → BitLocker Drive Encryption
Activar cifrado
1
Selecciona la unidad → Activar BitLocker
2
Elige método de autenticación: contraseña o USB
3
Guarda la clave de recuperación en un lugar seguro
🔥 Firewall de Windows DefenderGUI
Acceso
Seguridad de Windows → Firewall y protección de red
Permitir una aplicación
1
Clic en Permitir una aplicación a través del firewall
2
Selecciona la app y activa redes públicas o privadas

Ubuntu Herramientas de seguridad

🔥 UFW — CortafuegosCLI
sudo ufw enableActivar el cortafuegos
sudo ufw allow sshPermitir SSH
sudo ufw deny 80Bloquear puerto 80
sudo ufw statusVer reglas activas
🔒 Cifrado de disco (ecryptfs)CLI
Cifrar una carpeta específica
sudo apt install ecryptfs-utils
ecryptfs-setup-privateCifra la carpeta del usuario
💡 Alternativa: Durante la instalación de Ubuntu puedes habilitar el cifrado completo del disco desde el instalador.
🛡️ AppArmorCLI
¿Qué hace?
Restringe las capacidades de cada aplicación. Si una app es comprometida, no puede acceder a lo que no tiene permitido.
sudo aa-statusVer estado de AppArmor
🔑 Autenticación con claves SSHCLI
ssh-keygenGenera par de claves pública/privada
ssh-copy-id usuario@servidorCopia la clave pública al servidor
🔄 Actualizaciones automáticasCLI
sudo apt install unattended-upgrades
sudo dpkg-reconfigure unattended-upgrades
