6.1 Permisos de red y recursos compartidos

Controla quién accede a carpetas, impresoras y equipos remotos en la red.

Niveles de permiso disponibles

👁️
Lectura
Solo puede ver y abrir archivos.
✏️
Lectura y escritura
Ver, crear y modificar archivos.
🔑
Control total
Leer, escribir, ejecutar y eliminar.

Ubuntu Permisos y recursos compartidos

📁 Compartir con SambaCLI
1. Instalar Samba
sudo apt install samba
2. Crear la carpeta a compartir
mkdir /home/usuario/compartida
3. Editar configuración de Samba
sudo nano /etc/samba/smb.conf
[Compartida] path = /home/usuario/compartida read only = no browsable = yes
4. Añadir usuario y reiniciar
sudo smbpasswd -a usuarioAñade usuario a Samba
sudo systemctl restart smbdReinicia el servicio
🔒 Conexión remota: SSHCLI
Instalar y conectar
sudo apt install openssh-server
ssh usuario@ip_del_servidorDesde otro equipo
🖥️ Conexión remota: VNC (escritorio gráfico)
Instalar servidor VNC
sudo apt install tightvncserver
vncserverInicia el servidor
Cliente VNC: usa RealVNC o TigerVNC desde otro equipo para conectarte al escritorio gráfico del servidor Ubuntu.
📁 Samba: GUIGUI
Configuración → Recursos compartidos → selecciona carpeta → activa Compartir esta carpeta → define Lectura o Lectura y escritura.
