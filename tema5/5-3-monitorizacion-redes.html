5.3 Monitorización de redes

Supervisar el estado y rendimiento de la red para detectar problemas antes de que afecten a los usuarios.

¿Qué debemos monitorizar?

📊
Rendimiento: ancho de banda, latencia, jitter, pérdida de paquetes.
🟢
Disponibilidad (uptime): routers, switches, servidores activos.
🔍
Detección de fallos: conexiones caídas, enlaces críticos.
🔒
Seguridad: accesos no autorizados, tráfico anómalo (IDS/IPS).
📶
Análisis de tráfico: qué apps o dispositivos consumen más ancho de banda.
📅
Historial: tendencias a lo largo del tiempo para planificación.
🔔
Alertas: notificaciones automáticas cuando se superan umbrales críticos.

Windows Herramientas de monitorización

Herramientas integradasGUI
Administrador de tareas
Monitorización básica del rendimiento de red en tiempo real.
Acceso: Ctrl + Shift + Esc → pestaña Rendimiento
Monitor de recursos
Procesos usando la red, conexiones TCP activas, latencias.
Acceso: Administrador de tareas → Rendimiento → Abrir monitor de recursos
Visor de eventos
Logs de conectividad, caídas de red, problemas DNS y seguridad.
Acceso: Win + X → Visor de eventos
Herramientas de terceros
SolarWinds NPM
Monitorización avanzada: análisis de tráfico en profundidad, informes detallados de rendimiento.
Wireshark
Captura y análisis de paquetes de red. Útil para resolver problemas complejos de red y seguridad.

Ubuntu Herramientas de monitorización

Línea de comandosCLI
ifconfigEstado de interfaces (clásico)
ip aInterfaces de red (moderno)
netstatConexiones activas y estadísticas
ss -tulnSockets y puertos activos (más rápido)
nloadAncho de banda en tiempo real
sudo iftopTráfico de red por host
sudo wiresharkCaptura y análisis de paquetes
Instalar herramientas
sudo apt install nload
sudo apt install iftop
sudo apt install wireshark
Interfaz gráficaGUI
Monitor del Sistema
Gráficos en tiempo real de uso de red, tráfico entrante/saliente y procesos activos en la red.
Acceso: Menú de aplicaciones → Monitor del Sistema
Wireshark (GUI)
Captura y análisis de paquetes con interfaz visual intuitiva. Ideal para resolver problemas complejos.
Acceso: Menú aplicaciones → Wireshark
