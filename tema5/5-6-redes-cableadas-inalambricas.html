5.6 Redes cableadas e inalámbricas

Gestión de conexiones Ethernet (cableadas) y Wi-Fi (inalámbricas) en Windows y Ubuntu.

🔌 Red cableada (Ethernet)📶 Red inalámbrica (Wi-Fi)
VelocidadMayor y más estableVariable según señal
MovilidadLimitada (requiere cable)Alta (sin cables)
Config inicialAutomática por DHCPRequiere autenticación
InterferenciasSin interferenciasPosibles interferencias de radio

Windows Gestión de redes

🔌 Red cableada
Configuración manual (si DHCP no funciona)
1
Configuración → Red e Internet → Ethernet
2
Selecciona la red → Editar
3
Cambia Automático (DHCP) a Manual
4
Introduce IP, máscara, gateway y DNS
PowerShell — IP estática
Listar interfaces:
Get-NetAdapter
New-NetIPAddress -InterfaceAlias "Ethernet" -IPAddress 192.168.1.100 -PrefixLength 24 -DefaultGateway 192.168.1.1
📶 Red inalámbrica (Wi-Fi)
Conexión a una red Wi-Fi
1
Configuración → Red e Internet → Wi-Fi
2
Selecciona la red deseada → Conectar
3
Introduce la contraseña
Gestión de redes guardadas
Configuración → Wi-Fi → Administrar redes conocidas
CMD — conectar por nombre
netsh wlan connect name="MiRed"

Ubuntu Gestión de redes

🔌 Red cableada
Configuración manual
1
Configuración → Red
2
Selecciona Ethernet → icono de engranaje
3
Método: de Automático a Manual
4
Introduce IP, máscara, gateway y DNS → Aplicar
nmcli — IP estática
Ver interfaces:
nmcli device status
sudo nmcli con mod "Ethernet" ipv4.addresses 192.168.1.100/24 ipv4.gateway 192.168.1.1 ipv4.dns 8.8.8.8 ipv4.method manual
Activar conexión:
sudo nmcli con up "Ethernet"
📶 Red inalámbrica (Wi-Fi)
Conexión desde GUI
1
Icono de red → Wi-Fi
2
Elige la red → Conectar
3
Introduce la contraseña
nmcli — conexión por terminal
Ver redes disponibles:
nmcli device wifi list
nmcli device wifi connect "MiRed" password "Contraseña"
Redes guardadas
Configuración → Wi-Fi → Redes conocidas para editar o eliminar perfiles guardados.
