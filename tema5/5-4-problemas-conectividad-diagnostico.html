5.4 Problemas de conectividad y diagnóstico

Pasos y herramientas para identificar y resolver problemas de red de forma sistemática.

¿Qué revisar primero?
🔌
Hardware: cables, routers, switches, tarjetas de red.
⚙️
Config de red: IP, gateway, DNS, proxy. Detecta IPs duplicadas.
🛡️
Software: firewall mal configurado, antivirus bloqueando tráfico.
📋
Logs: revisar registros del router, switch o servidor.
Problemas comunes
❌ DNS fallido
No resuelve nombres de dominio → no carga la web.
❌ DHCP sin IP
El dispositivo no obtiene dirección → sin conexión.
❌ Latencia alta
Red lenta o inestable, ping elevado.
❌ Wi-Fi débil
Interferencias, señal baja o config incorrecta.

Windows Herramientas de diagnóstico

Comandos esencialesCLI
ping
Verifica conectividad enviando paquetes ICMP al destino.
ping www.google.com
tracert
Muestra cada salto hasta el destino. Detecta dónde se interrumpe la conexión.
tracert www.google.com
ipconfig
Ver config de red actual. Renovar dirección IP desde DHCP.
ipconfig /all · ipconfig /release · ipconfig /renew
nslookup
Diagnostica problemas DNS. Consulta si un nombre de dominio se resuelve correctamente.
nslookup www.google.com
Herramientas gráficasGUI
🔧 Solucionador de problemas
Detecta automáticamente: cables desconectados, IP incorrecta, adaptadores con fallos.
Clic derecho en icono de red → Solucionar problemas
📋 Visor de eventos
Revisa logs de red: errores de conectividad, DNS, seguridad.
Win + X → Visor de eventos → Registros de Windows → Sistema

Ubuntu Herramientas de diagnóstico

Comandos esencialesCLI
ping www.google.comVerificar conectividad
traceroute www.google.comTrazar ruta hasta destino
ifconfigConfig de interfaces (clásico)
ip aConfig de interfaces (moderno)
nslookup www.google.comDiagnóstico DNS
dig www.google.comDiagnóstico DNS avanzado
nmcli dev statusEstado de interfaces (NetworkManager)
tail -f /var/log/syslogLogs del sistema en tiempo real
Herramienta gráficaGUI
🌐 Network Manager
Revisar config de red, verificar conectividad, reconectar a una red o reconfigurar una conexión existente.
Icono de red (barra superior) → Configuración de red
