Ubuntu Herramientas de seguridad

UFW — CortafuegosCLI
Comandos básicos
sudo ufw allow sshPermitir SSH (puerto 22)
sudo ufw enableActivar el cortafuegos
sudo ufw statusVer reglas activas
sudo ufw deny 80Bloquear puerto 80
iptables — reglas avanzadas
sudo iptables -P INPUT DROP sudo iptables -A INPUT -p tcp --dport 22 -j ACCEPT
Efecto: bloquea todo el tráfico entrante excepto SSH.
OpenSSH y TLS/SSLCLI
Instalar y configurar OpenSSH
sudo apt install openssh-server
sudo nano /etc/ssh/sshd_config
# Líneas a añadir/modificar: PermitRootLogin no PasswordAuthentication no
sudo systemctl restart ssh
Conectarse con clave pública
ssh -i /ruta/clave_privada usuario@servidor
Certificados SSL con Certbot (TLS)
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx
