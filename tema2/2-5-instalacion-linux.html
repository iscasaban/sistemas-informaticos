2.5 Instalación de Linux

Proceso paso a paso para instalar Ubuntu Desktop

🐧 Ubuntu Desktop
¿Por qué Ubuntu? Es la distribución más accesible para iniciarse en Linux. Código abierto, gratuita y con una gran comunidad de soporte.

📋 Requisitos previos — ¿Qué necesitas?

💿

Medio de instalación

USB (mín. 4 GB) o DVD con la imagen ISO de Ubuntu.

🗂️

Copia de seguridad

La instalación puede borrar todos los archivos del disco.

🌐

Acceso a internet

Opcional pero recomendado para descargar actualizaciones.

🐧 Proceso de instalación — 15 pasos

1

Descargar Ubuntu GUI

  • Ve a ubuntu.com/download.
  • Elige Ubuntu Desktop y descarga el archivo .ISO.
2

Crear medio de instalación GUI

📱 Para USB:
Usa Rufus o Etcher.
Selecciona el .ISO → clic en Iniciar.
📀 Para DVD:
Usa ImgBurn o Brasero.
Graba el .ISO en el DVD.
3

Preparar la BIOS GUI

  • Reinicia y pulsa F2, F10, Supr o Esc.
  • En Boot, pon el USB/DVD en primera posición.
  • Guarda y reinicia.
4

Iniciar el instalador GUI

  • Aparece el logo de Ubuntu y un menú.
  • Elige Install Ubuntu.
  • 💡 Opcional: elige Try Ubuntu without installing para probar antes.
5

Idioma y teclado GUI

  • Selecciona el idioma de instalación → Continuar.
  • Elige la distribución del teclado (Ubuntu la detecta automáticamente).
6

Preparar la instalación GUI

  • Marca Descargar actualizaciones e instalar software de terceros (códecs y drivers).
  • Haz clic en Continuar.
7

Tipo de instalación GUI

🗑️ Borrar disco e instalar Ubuntu

Instalación limpia. Borra todo el contenido del disco.

🔀 Instalar junto a otro SO (Dual Boot)

Conserva Windows u otro SO. Arranca dual al encender.

⚙️ Más opciones (particiones personalizadas)

Control total sobre las particiones. Para usuarios avanzados.

8

Configurar particiones (avanzado) GUI

Partición Formato Tamaño recomendado Uso
/ (root) EXT4 20 GB mínimo Sistema de archivos principal
swap swap 2 – 8 GB (según RAM) Memoria virtual de intercambio

💡 Si no tienes experiencia, usa las opciones predeterminadas de Ubuntu.

9

Seleccionar zona horaria GUI

  • El instalador detecta tu ubicación automáticamente.
  • Corrígela en el mapa si es incorrecta → Continuar.
10

Crear cuenta de usuario GUI

  • Introduce nombre de usuario y contraseña (será la cuenta de administrador).
  • Elige entre Iniciar sesión automáticamente o Solicitar contraseña.
11

Instalación del sistema GUI

  • Ubuntu copia archivos y configura el sistema (varios minutos).
  • Se muestran diapositivas informativas durante el proceso.
12

Finalizar la instalación GUI

  • Aparece mensaje de instalación completada.
  • Retira el USB/DVD cuando se indique y pulsa Enter para reiniciar.
13

Primer arranque GUI

  • Aparece la pantalla de inicio de sesión de Ubuntu.
  • Introduce tu usuario y contraseña.
14

Actualizar el sistema GUI CLI

  • Por GUI: Configuración → Software y actualizaciones → Buscar actualizaciones.
  • Por terminal:
sudo apt update && sudo apt upgrade
  • Instala también controladores adicionales desde Software y actualizaciones → Controladores adicionales.
15

Instalar software adicional GUI CLI

  • Por GUI: usa la tienda Ubuntu Software.
  • Por terminal:
sudo apt install nombre_del_programa

⚖️ Comparativa rápida: Windows vs Ubuntu

Aspecto 🪟 Windows 10 🐧 Ubuntu
Licencia Privativa (clave de producto) Código abierto (gratuita)
Pasos de instalación 7 pasos 15 pasos
Medio recomendado USB (Media Creation Tool) USB con Rufus / Etcher
Partición principal NTFS EXT4
Instalar software Descarga manual / Microsoft Store sudo apt install / Ubuntu Software
Dual Boot Posible (más complejo) Opción integrada en el instalador
💡 Recuerda: Ubuntu permite probar el sistema sin instalar (Try Ubuntu) antes de comprometerse con la instalación. Ideal para comprobar compatibilidad de hardware.
