INTRO

¿Qué es un Sistema Operativo?

Definición clave: Un SO es un software de sistema que gestiona los recursos de hardware y software de un ordenador, actuando de intermediario entre el usuario y el hardware.

🔺 Capas de un sistema informático

👤 Usuario
📦 Aplicación
⚙️ Sistema Operativo
🖥️ Hardware

Cada capa se apoya en la de abajo. El SO es el puente central.

🛠️ Funciones principales del SO

⚡

Gestión de procesos

Planifica y coordina la ejecución de programas.

🧠

Gestión de memoria

Asigna y libera RAM. Usa memoria virtual.

💾

Gestión de almacenamiento

Organiza archivos y controla el acceso al disco.

🖨️

Gestión de dispositivos

Comunica el hardware mediante controladores (drivers).

🖱️

Interfaz de usuario

CLI (terminal) o GUI (ventanas e iconos).

🔒

Seguridad

Autenticación, cifrado y control de acceso.

2.1

Evolución histórica de los sistemas operativos

Idea clave: Los SO han pasado de simples programas de control (años 50) a complejas plataformas multifuncionales que gestionan hardware, redes, móviles y la nube.
1950 – 1960

Los primeros SO

  • Procesamiento por lotes: trabajos secuenciales en mainframes, sin interacción de usuario.
  • Tiempo compartido: varios usuarios, un ordenador. Ej.: CTSS (MIT, 1961).
  • Primer SO reconocido: GM-NAA I/O (1956).
1970

Unix y las redes

  • Unix (1969) – Bell Labs (Thompson & Ritchie). Sistema jerárquico de archivos, escrito en C. Base de muchos SO modernos.
  • Aparecen los SO de red (ARPANET → precursora de internet).
1980

La era del PC

  • MS-DOS (1981) · Microsoft · línea de comandos, estándar del PC.
  • macOS (1984) · Apple Macintosh · primera GUI popular: ratón y ventanas.
  • Unix deriva en BSD y Solaris.
1990

Windows y Linux

  • Windows 3.0 / 95 · GUI avanzada, Plug&Play · domina el mercado doméstico.
  • Linux (1991) · Linus Torvalds · código abierto basado en Unix · comunidad activa.
2000

Móviles y la nube

  • Windows XP (2001) · uno de los más populares de la historia.
  • macOS X (2001) · basado en Unix · ecosistema Apple.
  • iOS (2007) y Android (2008) · revolución móvil.
  • Nube: AWS, Google Cloud.
2010 – Actualidad

Convergencia y nuevas tecnologías

  • Windows 10 (2015) · integración móvil, Cortana, PC + tablet + consola.
  • Linux / Open Source · Ubuntu, CentOS, Red Hat · domina servidores e infraestructura.
  • IoT: Google Fuchsia, Amazon FreeRTOS para dispositivos inteligentes.
