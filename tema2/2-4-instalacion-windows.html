2.4 Instalación de Windows

Proceso paso a paso para instalar Windows 10

🪟 Windows 10
Nota: El proceso está pensado para Windows 10. Las versiones posteriores son muy similares. Siempre haz copia de seguridad antes de comenzar.

📋 Requisitos previos — ¿Qué necesitas?

💿

Medio de instalación

DVD o USB con los archivos de instalación de Windows.

🔑

Clave de producto

Necesaria para activar Windows (software propietario).

🌐

Conexión a internet

Opcional pero recomendable para descargar actualizaciones.

🗂️

Copia de seguridad

La instalación puede borrar archivos del disco.

🪟 Proceso de instalación — 7 pasos

1

Preparar la BIOS GUI

  • Enciende el PC y pulsa repetidamente F2, F10, Supr o Esc (según fabricante).
  • Ve a la sección Boot (arranque).
  • Coloca el USB o DVD en primera posición de arranque.
  • Guarda y reinicia.
2

Comenzar la instalación GUI

  • Aparece el logo de Windows → pulsa cualquier tecla.
  • Selecciona idioma, hora/moneda y teclado → Siguiente.
  • Haz clic en Instalar ahora.
  • Introduce la clave de producto si se solicita.
3

Elegir tipo de instalación GUI

  • Acepta los términos de licencia.
  • Elige Personalizada: instalar solo Windows (avanzado).
  • Selecciona el disco destino. Si es necesario, elimina particiones y usa el espacio no asignado.
4

Configurar particiones GUI

  • Si no hay partición, haz clic en Nuevo para crearla.
  • Selecciona la partición y haz clic en Siguiente.
  • La instalación comienza. El sistema se reiniciará varias veces.
5

Configuración inicial GUI

  • Selecciona región, idioma del teclado y red Wi-Fi (opcional).
  • Introduce nombre de usuario y contraseña para la cuenta principal.
  • Configura Cortana y servicios de personalización (opcional).
  • Elige configuración de privacidad y seguridad.
6

Activar Windows y actualizar GUI

  • Ve a Configuración → Actualización y seguridad → Activación e introduce la clave.
  • Ve a Configuración → Windows Update → Buscar actualizaciones.
  • Instala todas las actualizaciones disponibles.
7

Instalar controladores y software GUI

  • Descarga los controladores del hardware desde la web del fabricante.
  • Instala aplicaciones esenciales: navegador, ofimática, etc.
