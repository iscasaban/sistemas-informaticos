4.6 Monitorización de un sistema operativo

Observar y registrar el rendimiento y la actividad del sistema en tiempo real

¿Por qué monitorizar?

🚨
Prevención
Detectar anomalías antes de que se conviertan en fallos críticos
⚡
Rendimiento
Optimizar el uso de CPU, RAM y disco ajustando configuraciones
🛡️
Seguridad
Detectar accesos no autorizados o intentos de ataque
✅
Disponibilidad
Garantizar que los servicios estén siempre operativos
📊
Planificación
Anticipar necesidades futuras de almacenamiento o hardware

Métricas clave a vigilar

🖥 CPU 🧠 Memoria RAM 💾 Disco 🌐 Red ⚙️ Procesos activos
Windows 10 Herramientas de monitorización
📊 Administrador de tareas
Gráfico
Acceso: Ctrl + Shift + Esc
Pestaña Rendimiento → CPU, memoria, disco, red y GPU en tiempo real
Uso: visión rápida del estado general del sistema
🔬 Monitor de recursos
Gráfico
Desde Administrador de tareas → Rendimiento → Abrir monitor de recursos
→ detalla el consumo por proceso individual: CPU, disco, memoria y red
Uso: identificar qué proceso concreto está causando un problema
📈 Monitor de rendimiento
Gráfico
Acceso: Win + R → escribe perfmon
→ gráficos y datos históricos de CPU, disco, red y tiempo de respuesta
Uso: análisis profundo y detección de patrones a largo plazo
📋 Visor de eventos
Gráfico
Acceso: Win + X → Visor de eventos
→ errores, advertencias y eventos de auditoría de seguridad
Uso: detectar fallos, problemas de estabilidad y auditar seguridad
Ubuntu Herramientas de monitorización
⚙️ top / htop
Terminal
→ CPU, memoria y procesos en tiempo real:
top
→ Versión mejorada e interactiva:
sudo apt install htop && htop
htop permite matar procesos directamente desde la interfaz
📊 vmstat / iostat / netstat
Terminal
→ CPU, memoria, I/O y actividad del sistema
vmstat
→ Rendimiento de disco y almacenamiento:
iostat
→ Conexiones de red activas y tráfico:
netstat
iostat requiere instalar el paquete sysstat
🖥 Monitor del sistema (GNOME)
Gráfico
Acceso: menú de aplicaciones → Monitor del sistema
→ gráficos en tiempo real de CPU, memoria, intercambio y red
→ lista de procesos con opción de finalizarlos
Uso: equivalente al Administrador de tareas de Windows
