4.5 Registros y logs

Archivos donde el sistema guarda un historial de todo lo que ocurre

¿Para qué sirven los logs?

🔍
Diagnóstico
Identificar errores y fallos en servicios o aplicaciones
🛡️
Auditoría de seguridad
Detectar accesos sospechosos o brechas de seguridad
📈
Rendimiento
Identificar cuellos de botella o uso excesivo de recursos
⚖️
Cumplimiento normativo
Muchas regulaciones exigen conservar registros de actividad

Aspectos clave en la gestión de logs

💾 Almacenamiento
Guardar los logs de forma eficiente para no sobrecargar el sistema
👁 Monitoreo
Seguimiento en tiempo real para detectar incidentes rápido
🔄 Rotación
Archivar o eliminar logs antiguos con políticas automáticas
🏢 Centralización
Reunir logs de varios servidores en un único lugar
Windows 10 Visor de eventos
📋 Acceso y navegación
Gráfico
Acceso: Win + X → Visor de eventos
Panel izquierdo → categorías de registros:
Registros de Windows: Aplicación, Seguridad, Configuración, Sistema
Registros de apps y servicios: eventos específicos de cada aplicación
🔎 Trabajar con eventos
Gráfico
→ cada evento muestra hora, origen y nivel (info / advertencia / error)
→ filtrar por nivel o buscar eventos concretos
→ Guardar eventos como… para archivar
→ Borrar registro para liberar espacio
Auditoría de seguridad: Registros de Seguridad → inicios de sesión, cambios de configuración, accesos
Ubuntu Directorio /var/log
📁 Logs importantes
/var/log/syslogregistros generales del sistema
/var/log/auth.logautenticaciones e inicios de sesión
/var/log/dmesgmensajes del kernel (hardware)
/var/log/kern.logmensajes específicos del kernel
🔎 Leer y monitorizar logs
Terminal
Listar archivos de log:ls /var/log
Ver contenido completo:cat /var/log/syslog
Ver las últimas líneas:tail /var/log/syslog
Monitorizar en tiempo real:tail -f /var/log/syslog
🔄 Rotación y centralización
Terminal
Rotar logs manualmente:
sudo logrotate -f /etc/logrotate.conf
logrotate: archiva, comprime y elimina logs antiguos automáticamente según /etc/logrotate.conf

rsyslog: centraliza logs de múltiples servidores en uno
