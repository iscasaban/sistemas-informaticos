4.3 Seguridad de un sistema operativo

Proteger la información, garantizar la estabilidad y evitar accesos no autorizados

Mecanismos clave de seguridad

🔑
Control de acceso
Solo usuarios autorizados acceden a archivos, apps y redes
🔄
Actualizaciones
Parches que corrigen vulnerabilidades conocidas
🛡️
Antivirus / antimalware
Protección contra virus y software malicioso
🔥
Cortafuegos
Filtra el tráfico de red entrante y saliente
🔒
Encriptación
Los archivos son ilegibles sin la clave de acceso
Windows 10 Herramientas de seguridad
🛡️ Windows Defender
Gráfico
Configuración → Actualización y seguridad → Seguridad de Windows
→ Protección contra virus y amenazas
Función: protección en tiempo real contra malware y virus
🔥 Cortafuegos de Windows
Gráfico
Panel de control → Sistema y seguridad → Cortafuegos de Windows
Función: permitir o bloquear apps y establecer reglas de red
🔒 BitLocker
Gráfico
Panel de control → Sistema y seguridad → Cifrado de unidad BitLocker
Función: cifra el disco completo; sin la clave nadie puede acceder a los datos
👤 Cuentas y autenticación
Gráfico
Configuración → Cuentas
→ Contraseña fuerte, PIN o autenticación de dos factores
→ Desactivar cuentas de invitado o no usadas
🔄 Windows Update
Gráfico
Configuración → Actualización y seguridad → Windows Update
Recomendación: mantener las actualizaciones de seguridad en automático
Ubuntu Herramientas de seguridad
🔄 Actualizar el sistema
Terminal
sudo apt update && sudo apt upgrade
→ instala las últimas versiones y parches de seguridad
🔥 UFW — cortafuegos
Terminal
sudo ufw enableactiva el cortafuegos
sudo ufw allow sshpermite conexiones SSH
sudo ufw statuscomprueba el estado
🛡️ ClamAV — antivirus (opcional)
Terminal
sudo apt install clamavinstala el antivirus
sudo freshclamactualiza base de datos de virus
sudo clamscan -r /homeanaliza el directorio home
🔒 Encriptación (LUKS / EncFS)
Terminal
Durante la instalación: cifrar disco completo con LUKS
Cifrar carpetas específicas:
sudo apt install gnome-encfs-manager
🚫 Desactivar servicios innecesarios
Terminal
Listar servicios activos:
sudo systemctl list-units --type=service
Desactivar un servicio innecesario:
sudo systemctl disable nombre_servicio
