1.5 Mapa físico y lógico de una red local

Dos formas complementarias de documentar y entender la infraestructura de una red LAN.

¿Por qué son necesarios? Permiten diseñar, implementar y gestionar redes de forma organizada. Facilitan el diagnóstico de problemas y la planificación de cambios.
🔌 Mapa físico
Muestra la disposición real del hardware: cables, puertos, ubicación de dispositivos.
Incluye: routers, switches, APs, PCs, impresoras, racks y paneles de conexión.
Especifica el tipo de cable: Cat5e, Cat6, fibra óptica…
Identifica los puertos de routers y switches para facilitar la gestión.
Componentes clave
RouterConexión con redes externas / internet
SwitchDistribuye la conexión a múltiples dispositivos
APPunto de acceso inalámbrico
Dispositivos finalesPCs, impresoras, cámaras…
Cableado estructuradoCat5e / Cat6, racks y paneles
Herramientas para crearlo
LucidchartOnline, fácil de usar
Draw.ioGratuito y versátil
Microsoft VisioProfesional, más completo
Cómo crearlo
1
Abre la herramienta y elige iconos de red (router, switch, PC…)
2
Representa cada dispositivo con su icono correspondiente
3
Conecta los dispositivos con líneas que representen cables físicos
🧩 Mapa lógico
Muestra cómo se comunican los dispositivos: IPs, subredes, protocolos.
Incluye: direcciones IP (estáticas o DHCP), rangos de subred y VLANs.
Describe cómo se enrutan los datos dentro de la red.
Configura la seguridad Wi-Fi: SSID, WPA2, WPA3.
Componentes clave
Direcciones IPEstáticas o dinámicas (DHCP)
SubredesEj: 192.168.1.0/24
VLANsSeparan tráfico y mejoran la seguridad
ProtocolosTCP/IP, ARP, ICMP, DHCP
Wi-FiSSID + seguridad WPA2 / WPA3
Herramientas para crearlo
Packet TracerSimulador de redes de Cisco (gratuito)
SolarWinds NTMNetwork Topology Mapper profesional
Draw.ioTambién válido para mapas lógicos
Cómo crearlo
1
Representa los dispositivos con sus direcciones IP asignadas
2
Incluye subredes con su rango IP, ej. 192.168.1.0/24
3
Agrega rutas estáticas o dinámicas entre subredes

Comparativa: mapa físico vs mapa lógico

Aspecto🔌 Mapa físico🧩 Mapa lógico
¿Qué muestra? Disposición real del hardware Configuración y flujo de datos
Componentes Routers, switches, cables, racks IPs, protocolos, subredes, VLANs
Responde a… ¿Dónde está cada dispositivo? ¿Cómo se comunican los dispositivos?
Herramientas Visio, Draw.io, Lucidchart Packet Tracer, SolarWinds, Draw.io
Útil para… Instalación y mantenimiento físico Diagnóstico de red y optimización
💡 Clave: ambos mapas son complementarios, no excluyentes. El físico asegura una infraestructura bien organizada; el lógico permite gestionar eficazmente el flujo de datos.
