1.2 Estructura y componentes

Los elementos físicos de un sistema informático y cómo trabajan juntos.

🔩 Placa base

La columna vertebral del ordenador. Conecta y permite la comunicación entre todos los componentes.

Puertos E/SUSB, HDMI, Ethernet, audio…

🧠 Procesador (CPU)

El cerebro del sistema. Ejecuta instrucciones y realiza cálculos. Arquitecturas modernas: x86 (Intel/AMD) y ARM (móviles).
🎛️ Unidad de Control (CU)
Interpreta y ejecuta las instrucciones del programa.
➕ ALU
Operaciones matemáticas (suma, resta) y lógicas (AND, OR).
📌 Registros
Memoria ultrarrápida dentro de la CPU para datos temporales.
🔢 Contador de programa
Guarda la dirección de la próxima instrucción a ejecutar.
📋 Set de instrucciones
Conjunto de operaciones básicas que puede realizar la CPU.
🔔 Interrupciones
Pausan la CPU para atender eventos externos (teclado, red…).

💾 Memoria interna

TipoCaracterísticaVolatilidadEjemplos
RAM Almacenamiento temporal de datos en uso Volátil DDR4, DDR5, LPDDR
ROM / xPROM Datos permanentes (firmware, BIOS) No volátil ROM, EPROM, EEPROM
Caché Memoria ultrarrápida integrada en la CPU Volátil L1, L2, L3
💡 Volátil = se borra al apagar el equipo  ·  No volátil = los datos persisten sin corriente.
