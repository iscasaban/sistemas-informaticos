3.1 Gestión de sistemas mediante comandos

Dos herramientas principales: PowerShell (Windows) y Bash (Ubuntu/Linux)

Windows PowerShell
📁 Sistema de archivos
Get-ChildItem listar archivos y carpetas
Set-Location cambiar de directorio
⚙️ Procesos
Get-Process ver procesos activos
Stop-Process finalizar un proceso
👤 Usuarios
Get-LocalUser listar usuarios locales
New-LocalUser crear nuevo usuario
🤖 Scripts
.\script.ps1 ejecutar un script
Linux Bash (Ubuntu)
📁 Sistema de archivos
ls listar contenido de un directorio
cd cambiar de directorio
⚙️ Procesos
ps aux ver procesos activos
kill -9 finalizar un proceso por ID
👤 Usuarios
whoami ver usuario actual
sudo adduser crear nuevo usuario
🤖 Scripts
./script.sh ejecutar un script
Scripts PowerShell: extensión .ps1 — ideales para limpiar archivos temporales y registros de eventos de forma automática.
Scripts Bash: extensión .sh — muy usados para copias de seguridad con rsync. Necesitan permisos con chmod +x.
