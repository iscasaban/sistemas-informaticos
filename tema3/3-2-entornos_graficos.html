3.2 Gestión de sistemas mediante entornos gráficos

Herramientas visuales para administrar el sistema sin usar la terminal

Windows Herramientas gráficas
🗂 Panel de control
Gestión de dispositivos, red, energía y hardware. Acceso desde el menú de inicio.
Ejemplo: activar o desactivar el Firewall de Windows Defender
⚙️ Configuración Win + I
Personalización, cuentas de usuario, privacidad y permisos de apps.
Ejemplo: conectarse a una red Wi-Fi
📊 Administrador de tareas Ctrl + Shift + Esc
Supervisar procesos activos y uso de CPU, memoria y disco.
Ejemplo: finalizar un programa que no responde
🔌 Administrador de dispositivos
Gestionar hardware: instalar, actualizar controladores y diagnosticar problemas.
Ejemplo: actualizar el controlador de red
Ubuntu Herramientas gráficas (GNOME)
⚙️ Configuración del sistema
Gestiona dispositivos de hardware, controladores y periféricos del sistema.
Ejemplo: actualizar un controlador de red
📊 Monitor del sistema
Supervisa uso de CPU, memoria y disco. Permite finalizar apps que no responden.
Ejemplo: cerrar un proceso que consume demasiados recursos
🔄 Gestor de actualizaciones
Instala actualizaciones de software y sistema. Aparece automáticamente o búscalo como "Actualizaciones de software".
Ejemplo: aplicar actualizaciones disponibles
💾 Utilidad de discos
Administra particiones, monta o desmonta discos. Búscalo como "Discos" en el menú.
Ejemplo: crear y formatear una nueva partición
