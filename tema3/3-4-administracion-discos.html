3.4 Administración de discos

Organizar, mantener y optimizar el almacenamiento en dispositivos físicos y virtuales

Windows Herramientas de disco
🖥 Administrador de discos Gráfico
Interfaz visual para gestionar particiones y volúmenes. Acceso con Win + X o escribiendo diskmgmt.msc.
✔ Crear, eliminar y formatear particiones
✔ Cambiar la letra de unidad
✔ Reducir o extender volúmenes
Ejemplo: clic derecho en espacio no asignado → Nuevo volumen simple → sigue el asistente
💻 DiskPart Terminal
Herramienta de línea de comandos para tareas avanzadas. Escribe diskpart en PowerShell como administrador.
list disklistar discos
list volumelistar volúmenes
select disk [n]seleccionar disco
cleaneliminar todas las particiones
create partition primarycrear partición
format fs=ntfs quickformatear en NTFS
assign letter=[X]asignar letra de unidad
Ubuntu Herramientas de disco
🖥 Utilidad de discos (GNOME Disks) Gráfico
Herramienta visual accesible desde el menú buscando "Discos". Intuitiva para tareas básicas.
✔ Crear, eliminar y redimensionar particiones
✔ Formatear discos con distintos sistemas de archivos
✔ Configurar puntos de montaje automáticos
Formatear: selecciona disco → icono engranaje → Formatear disco → elige sistema de archivos
💻 fdisk / parted / lsblk Terminal
Herramientas avanzadas para particionado desde la terminal.
lsblkver discos y particiones
sudo fdisk /dev/sdXparticionar un disco
nnueva partición (dentro de fdisk)
deliminar partición
wguardar cambios
sudo parted /dev/sdXpara discos grandes
Montar partición:
sudo mkdir /mnt/mi_particion
sudo mount /dev/sdX1 /mnt/mi_particion
Añadir a /etc/fstab para montaje automático al inicio
